# What have you done?!

An simple example of how to implement [Akita](https://netbasal.gitbook.io/akita/) a state management library into 
angular with communication to the backend crafted by [Nest.js](https://github.com/nestjs/nest/)

## Install

### Server
`cd server && npm i`

### Web
`cd web && npm i`

## Run

### Server
`cd server && npm start`

### Web
`cd web && npm start`

## Postscript
Have fun!
